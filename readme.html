プロジェクト構成

Assets > Resoruces > Models > marukC : 3Dモデルデータ

  • materials : サンプルのシェーダを設定したキャラクターのマテリアル
  • textures : キャラクターのテクスチャ
  • marukC : キャラクターの元データ
  • marukC_rig : キャラクターの元データ（Rig付き）
  • marukC_setup : マテリアルセットアップ済みのキャラクターの Prefab
  • marukC_rig_setup : マテリアルセットアップ済みのキャラクターの Prefab（Rig付き）
  • sample_wait.controller : キャラクターアニメーションのサンプル

Assets > Resoruces > Models > UnityChan : ユニティちゃん(https://unity-chan.com/)よりインポートしたデータ

  • Animations : アニメーションデータ
  • UCL2.0 : ユニティちゃんのライセンス規約

Assets > Resoruces > Shaders : シェーダファイル

  • CameraNormalTexture : 追加の法線情報を書き出す
  • PfxSample : サンプルのポストエフェクトシェーダー
  • SimpleToon : サンプルのトゥーンシェーダ

Assets > Scenes : サンプルシーンファイル

  • ModelViewer : サンプルシーン

Assets > Scripts : スクリプトファイル

  • CameraNormalTexturePass : 追加の法線テクスチャを書き出すパス
  • PfxRendererFeature : 追加のパスを管理する Renderer Feature 設定
  • PfxRenderPass : 追加のポストエフェクト(ポストプロセス)パス
  • PfxVolume : 追加のポストプロセスボリュームのスクリプト
  • SimpleCameraController : カメラ操作のスクリプト

Assets > Settings : レンダリング設定
